# Project X - VJTI: Replicating 1986 Backprop Paper

This repository contains an autonomous implementation of the Generalized Delta Rule (Backpropagation) derived from the 1986 Rumelhart, Hinton, & Williams paper.

## How to Run
1. Install Python 3
2. Install requirements: `pip install torch matplotlib`
3. Execute the training routine:
